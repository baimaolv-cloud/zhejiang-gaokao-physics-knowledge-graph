# 浙江新高考物理知识图谱 · 多版本交付

> 面向浙江新高考命卷视角的物理知识图谱产品。覆盖 **6 大模块、27 主知识点、123 细分考点、27 道真实浙江高考真题**，每考点配套 **易错 / 易混 / 妙法** 三维度，并内置 **25 种 Canvas 动态演示**。

## 版本总览

| 版本 | 文件 | 特点 | 适用场景 |
|------|------|------|----------|
| 在线交互版 | `physics_detailed_graph.html` | 考点树 + 真题 + 演示联动，零依赖 | 浏览器直接打开 |
| 纯演示版 | `physics_demos.html` | 25 种动态仿真单页 | 课堂投屏 |
| 练习册 | `physics_practice.md` | 27 真题 + 解析 | 打印 / 阅读 |
| 综合题突破 | `comprehensive.html` | 4 道真实浙江选考综合大题 + 对应示意图 + 结构化破题 | 专项突破训练 |
| 单文件版 | `物理知识图谱_单文件版.html` / `.zip` | 5 Tab 合一（含综合题突破），约 108KB 单文件 | 微信 / 邮件分享 |
| 离线版 | `物理知识图谱_离线版/` + `.zip` | 文件夹零依赖 | 无网环境 |
| 单机版 | `物理知识图谱_单机版/` + `.zip` | 文件夹零依赖 | U 盘携带 |

> 所有 HTML 版本均 **零外部依赖、无需联网**，双击即可运行。

## 快速开始

- 浏览器直接双击任意 `.html` 即可使用。
- 离线版 / 单机版：解压对应 `.zip` 后打开内部 `index.html`。
- 单文件版：可直接拖入浏览器，或作为单文件转发（无需任何附件）。

## 内容结构

- **6 模块**：力学 / 电磁 / 热学 / 光学 / 原子物理 / 实验
- **27 主知识点 · 123 细分 · 27 道浙江新高考真实真题**
- 每考点含：`易错(pit)` · `易混(con)` · `妙法(tip)` · `动态演示(demo)`
- **25 种 Canvas 仿真**：矢量合成、v-t 图、受力平衡、平抛、圆周运动、楞次定律、光电效应、干涉衍射等
- **综合题突破**（独立模块）：4 道真实浙江选考多知识点综合大题（力电磁综合 / 力学多过程 / 力热综合 / 实验设计），每题配对应示意图 + 结构化破题（考点定位 → 模型建构 → 分步求解 → 易错陷阱）

## 再生成（可选）

```bash
python gen_detailed.py   # 生成交互源 html / 演示页 / 练习册
python build_dist.py     # 打包离线版 + 单机版（含 zip）
python build_single.py   # 生成单文件版（含 zip）
```
